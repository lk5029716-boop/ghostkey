# App brief

**App name:** GhostKey  
**Tagline:** Your digital life. Passed on automatically.

**Brand:** Dark navy, amber key gold. Ghost motif.  
**Primary asset:** `/opt/data/GhostKey/logo.svg`

**Target platforms:** Flutter (Android + iOS)  
**Backend:** Supabase  
**Monetisation:** Freemium subscriptions via Stripe

**Visual direction:**
- Dark mode default
- Amber accent
- Key icon on lock screen + vault screens
- Status card: timer + contacts + vault state
